# SETTING SERVER SCUM v1.0.1-local

วันที่ปล่อย: 2026-04-21

เวอร์ชันนี้คือชุด local portable สำหรับคนที่อยากแตกไฟล์แล้วกดเปิดใช้งาน ไม่ต้องจำคำสั่ง `npm` เอง

## วิธีใช้แบบสั้น

1. แตกไฟล์ `SETTING-SERVER-SCUM-v1.0.1-local.zip`
2. ดับเบิลคลิก `Start SETTING SERVER SCUM.cmd`
3. เปิดหน้าเว็บแล้วไปที่ App Settings เพื่อตั้ง SCUM config folder, Nodes folder, Spawners folder และ backup folder

## ของที่เพิ่มใน release นี้

- Portable launcher ที่เช็ก Node.js, dependencies, port และเปิด browser ให้อัตโนมัติ
- Visual Diff ก่อน Save พร้อมสรุปความเสี่ยงว่าไฟล์ไหนเปลี่ยนและมี warning/critical ไหม
- Undo/history ต่อไฟล์ก่อน save สำหรับ Server Settings, Core Files และ Loot Studio
- Loot Setup Wizard สำหรับเลือกแนว solo, pvp, hardcore, bunker rich และ medical relief
- Item Catalog Pack เพิ่มชื่ออ่านง่าย หมวด rarity และ tags พร้อมรองรับ icon ที่มีอยู่
- Support bundle zip สำหรับส่งข้อมูล diagnostic/log แบบไม่ติด path ส่วนตัว
- Backup compare อ่านง่ายขึ้น ใช้เทียบ backup สองชุดหรือไฟล์เดียวได้
- Health Center แนะนำเป็นภาษาคนว่าควรแก้อะไรก่อน
- Performance smoke test กับ workspace ใหญ่จำลอง 160 nodes / 55 spawners

## ก่อนปล่อยตรวจอะไรแล้ว

รันผ่านคำสั่ง:

```bash
npm run release:quality
```

ชุดตรวจนี้รวม syntax check, release check, config roundtrip, sample workspace smoke, performance smoke, docs check, changelog check, portable package build และ Playwright regression tests
