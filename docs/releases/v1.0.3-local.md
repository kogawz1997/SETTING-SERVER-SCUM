# SETTING SERVER SCUM v1.0.3-local

วันที่ปล่อย: 2026-04-21

เวอร์ชันนี้แก้ release workflow ให้ใช้งานบน GitHub Actions ได้จริงขึ้น โดยเปลี่ยน lockfile ให้ใช้ public npm registry และให้ workflow อ่าน release notes หลัง checkout source แล้ว

## วิธีใช้สำหรับผู้ใช้ทั่วไป

1. ดาวน์โหลด `SETTING-SERVER-SCUM-v1.0.3-local.zip` จาก GitHub Release
2. แตก zip
3. ดับเบิลคลิก `Start SETTING SERVER SCUM.cmd`
4. ตั้ง path ใน App Settings แล้วรัน Preflight ก่อนแก้ไฟล์จริง

## จุดสำคัญ

- Portable launcher พร้อมใช้งาน
- Visual Diff ก่อน Save
- Undo/history ต่อไฟล์ก่อน save
- Loot Setup Wizard
- Item Catalog + icon support
- Support bundle zip
- Backup compare
- Health Center ภาษาคน
- GitHub Actions release workflow สำหรับสร้างและแนบ portable zip อัตโนมัติ

## ตรวจคุณภาพ

รันผ่าน:

```bash
npm run release:quality
```
