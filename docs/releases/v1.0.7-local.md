# SETTING SERVER SCUM v1.0.7-local

รอบนี้เป็น release เก็บฐานระบบฝั่ง local ให้แน่นขึ้น ก่อนเพิ่มฟีเจอร์ใหญ่รอบต่อไป

## ของใหม่

- แยก `workspace-domain` ให้เหลือหน้าที่ wiring เป็นหลัก
- เพิ่ม `loot-core-service` สำหรับ parser, validation, ref index และ loot analysis แกนกลาง
- เพิ่ม `workspace-file-service` สำหรับ resolve path, scan Nodes/Spawners, validate content และ diff
- เพิ่ม unit tests เฉพาะ service ใหม่
- เพิ่ม release guards กัน logic สำคัญไหลกลับเข้าไฟล์ใหญ่

## วิธีใช้

โหลด zip จาก Assets แตกไฟล์ แล้วเปิด `Start SETTING SERVER SCUM.exe` หรือใช้ `.cmd` fallback ถ้าเครื่องบล็อก exe

## หมายเหตุ

- ตัวนี้ยังเป็น local-first tool ไม่ใช่ SaaS
- ก่อนแก้ไฟล์จริงควรตั้ง backup path และรันหน้า readiness/diagnostics ให้ผ่านก่อน
