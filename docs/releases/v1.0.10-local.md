# SETTING SERVER SCUM v1.0.10-local

รอบนี้เป็น release สำหรับใช้งานจริงหลังเคลียร์หน้าเว็บ production ให้สะอาดขึ้น ไม่ให้หน้าเช็กงาน/หน้า internal โผล่ในเมนูหลักของผู้ใช้ทั่วไป

## ของใหม่

- เมนูหลักเหลือเฉพาะหน้าที่ใช้ทำงานจริง: Dashboard, App Settings, Server Settings, Core Files, Loot Studio, Analyzer, Graph, Profiles, Backups และ Help
- ถอดหน้า Customer Ready ออกจาก public route และ DOM แล้ว เพราะเป็นหน้าตรวจส่งมอบมากกว่าหน้าใช้งานประจำ
- ซ่อน Activity และ Diff Preview ออกจากเมนูหลัก เพื่อไม่ให้ผู้ใช้ทั่วไปสับสน
- ยังเก็บ Activity และ Diff Preview เป็น flow ภายในสำหรับ audit log, backup regression และ preview ก่อน save
- เพิ่ม UI smoke test เพื่อกันไม่ให้หน้า internal/test กลับมาโผล่ใน production navigation อีก
- อัปเดต release guard ให้เช็กว่า navigation production ไม่มีหน้า internal เหลืออยู่

## วิธีใช้

โหลด zip จาก Assets, แตกไฟล์ลงโฟลเดอร์ใหม่ แล้วเปิด `Start SETTING SERVER SCUM.exe`

ถ้า Windows บล็อกไฟล์ `.exe` ให้เปิด `Start SETTING SERVER SCUM.cmd` แทน

หลังเปิดเว็บ ให้เริ่มจาก Dashboard แล้วตรวจ Startup Doctor ก่อนชี้ path ไปยังไฟล์จริง

## หมายเหตุ

- ยังเป็น local-first tool ไม่ใช่ SaaS
- หน้า Activity ยังเปิดตรงผ่าน `/activity` ได้สำหรับตรวจ log
- หน้า Diff Preview ยังถูกเปิดจาก workflow preview/save ภายในแอป ไม่ได้เป็นหน้าที่ต้องกดจากเมนูหลัก
