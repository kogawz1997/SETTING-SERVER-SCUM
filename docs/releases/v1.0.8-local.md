# SETTING SERVER SCUM v1.0.8-local

รอบนี้เป็น local release สำหรับคนโหลด zip ไปใช้งานจริง เน้นทำให้ Loot Studio เข้าใจง่ายขึ้น, graph แก้ ref ได้จริงขึ้น, simulator อ่านผลได้ชัดขึ้น และ portable launcher ตรวจตัวเองก่อนเปิดแอป

## ของใหม่

- Graph relationship editor จัดการความสัมพันธ์ Spawner -> Node ได้จากหน้า Graph พร้อม stage/preview/apply
- Item Catalog เพิ่ม metadata แบบ curated สำหรับไอเทมสำคัญ เช่น กระสุน 12 gauge, กระเป๋า, เสื้อเกราะ, gold bar, fuel can และ absinthe
- Loot Simulator แสดง roll plan, expected category summary และ unresolved refs เพื่อให้เห็นว่าลูทจะออกจากกลุ่มไหนและอะไรยังอ้างผิด
- Portable package สร้าง `portable-manifest.json` ให้ launcher ใช้ตรวจไฟล์จำเป็นก่อนเริ่ม server
- Launcher เขียน header ลง `logs/startup.log` พร้อม version/path/node/port/url เพื่อ debug เครื่องลูกค้าได้ง่ายขึ้น
- GitHub release workflow ใช้ action รุ่นใหม่และ publish ผ่าน GitHub CLI บน runner

## วิธีใช้

โหลด zip จาก Assets, แตกไฟล์ลงโฟลเดอร์ใหม่ แล้วเปิด `Start SETTING SERVER SCUM.exe`

ถ้า Windows บล็อกไฟล์ `.exe` ให้ใช้ `Start SETTING SERVER SCUM.cmd` แทน

ถ้าเปิดแล้วแจ้งว่าไฟล์ขาด ให้แตก zip ใหม่และเทียบกับ `portable-manifest.json`

## หมายเหตุ

- ยังเป็น local-first tool ไม่ใช่ SaaS
- ก่อนแก้ไฟล์จริงให้ตั้งค่า path, backup folder และตรวจหน้า Customer Ready/Diagnostics ก่อน
- ถ้าโยกเครื่อง ให้ export/import workspace แล้ว remap path ใหม่ใน App Settings
