# SETTING SERVER SCUM v1.0.5-local

Release date: 2026-04-21

This release is the P2.13 local power-pack pass. It keeps the project local-first, but makes the everyday workflow easier for people who do not want to touch JSON by hand.

## For Normal Use

1. Download `SETTING-SERVER-SCUM-v1.0.5-local.zip` from GitHub Release.
2. Extract the zip.
3. Double-click `Start SETTING SERVER SCUM.exe`.
4. If Windows blocks the EXE, double-click `Start SETTING SERVER SCUM.cmd`.
5. Set paths in App Settings and run Preflight before editing real server files.

## Highlights

- Portable package now includes a real Windows EXE launcher when the build machine has either .NET SDK or the built-in .NET Framework C# compiler.
- Item Catalog uses the icon pack to infer readable English names, Thai labels, rarity, category, and tags.
- Loot presets now cover more common server styles such as solo, PvP, hardcore, bunker rich, survival poor, police station, starter friendly, vehicle support, and rare weapon low-spawn tuning.
- Simulator supports deterministic seeds and expected drop-rate summaries so draft vs saved comparisons are easier to trust.
- Graph focus panel can preview and apply spawner ref add/remove/replace operations through the normal backup and validation flow, including drag-to-connect from a spawner to a node/ref.
