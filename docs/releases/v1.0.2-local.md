# SETTING SERVER SCUM v1.0.2-local

วันที่ปล่อย: 2026-04-21

เวอร์ชันนี้คือ release สำหรับปิดงาน package จริงบน GitHub โดยเพิ่ม workflow ที่สร้าง GitHub Release และแนบ portable zip ให้อัตโนมัติเมื่อ push tag `v*-local`

## วิธีใช้สำหรับผู้ใช้ทั่วไป

1. ดาวน์โหลดไฟล์ `SETTING-SERVER-SCUM-v1.0.2-local.zip` จากหน้า GitHub Release
2. แตก zip
3. ดับเบิลคลิก `Start SETTING SERVER SCUM.cmd`
4. ไปที่ App Settings แล้วตั้ง SCUM config folder, Nodes folder, Spawners folder และ backup folder

## ของที่อยู่ในชุดนี้

- Portable launcher สำหรับเปิดแบบไม่ต้องพิมพ์ `npm`
- Visual Diff ก่อน Save พร้อม risk summary
- Undo/history ต่อไฟล์ก่อน save
- Loot Setup Wizard
- Item Catalog พร้อมชื่ออ่านง่าย หมวด rarity tags และ icon support
- Support bundle zip แบบ sanitize path
- Backup compare แบบอ่านง่าย
- Health Center ที่บอกว่าควรแก้อะไรก่อน
- Performance smoke coverage สำหรับ workspace ใหญ่จำลอง
- GitHub Actions workflow สำหรับสร้าง release zip และแนบไฟล์กับ GitHub Release

## ตรวจคุณภาพแล้ว

รันผ่าน:

```bash
npm run release:quality
```

และ workflow release จะรัน guard checks ฝั่ง GitHub อีกชุดก่อนสร้าง zip
