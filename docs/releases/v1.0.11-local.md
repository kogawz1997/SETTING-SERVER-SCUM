# SETTING SERVER SCUM v1.0.11-local

รอบนี้เป็น patch release สำหรับเคลียร์ release engineering ให้ตรงกันทุกจุดก่อนแจกใช้งานจริง

## ของใหม่

- เพิ่ม `LICENSE` แบบ MIT ที่ root repo
- เพิ่ม `license: MIT` ใน `package.json` และ `package-lock.json`
- อัปเดต README ให้ชี้ release ล่าสุดเป็น `v1.0.11-local`
- เพิ่ม release guard ให้เช็กว่า README, release note, changelog และ package version sync กัน
- ยืนยันว่า package scripts สำหรับ portable release มีครบ: `package:portable`, `package:portable:smoke`, และ `package:portable:zip-smoke`

## วิธีใช้

โหลด `SETTING-SERVER-SCUM-v1.0.11-local.zip` จาก Assets, แตกไฟล์ลงโฟลเดอร์ใหม่ แล้วเปิด `Start SETTING SERVER SCUM.exe`

ถ้า Windows บล็อกไฟล์ `.exe` ให้เปิด `Start SETTING SERVER SCUM.cmd` แทน

## หมายเหตุ

- ยังเป็น local-first tool ไม่ใช่ SaaS
- release นี้เน้น metadata, license และความเนียนของ repo/release ไม่ใช่เพิ่มฟีเจอร์ใหญ่
