# SETTING SERVER SCUM v1.0.4-local

วันที่ปล่อย: 2026-04-21

เวอร์ชันนี้แก้ release workflow ให้ผ่านบน GitHub Actions โดยทำให้ integration test สำหรับ import package ใช้ sample workspace ที่แน่นอนใน CI ไม่พึ่ง config ส่วนตัวในเครื่อง local

## วิธีใช้สำหรับผู้ใช้ทั่วไป

1. ดาวน์โหลด `SETTING-SERVER-SCUM-v1.0.4-local.zip` จาก GitHub Release
2. แตก zip
3. ดับเบิลคลิก `Start SETTING SERVER SCUM.cmd`
4. ตั้ง path ใน App Settings แล้วกด Run Preflight ก่อนแก้ไฟล์จริง

## จุดสำคัญ

- แก้ integration test ของ package import preview ให้รันได้ทั้งในเครื่อง local และ GitHub Actions
- ใช้ sample workspace สำหรับตรวจ flow import แบบ dry-run โดยไม่แตะไฟล์จริงของผู้ใช้
- ยังใช้ release quality gate ชุดเดิมก่อนสร้าง portable zip

## ตรวจคุณภาพ

ก่อนปล่อยต้องรันผ่าน:

```bash
npm run release:quality
```
